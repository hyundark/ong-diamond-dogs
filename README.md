# Diamond Dogs – ONG de Resgate Canino

**Projeto Final – Desenvolvimento Front-end para Web**  
**100% HTML5 + CSS3 + JavaScript**

Live demo: https://hyundark.github.io/ong-diamond-dogs/)


## Páginas desenvolvidas (exigência atendida)

| Página                | Arquivo             | Descrição |
|-----------------------|---------------------|---------|
| Página Inicial        | `index.html`        | Apresentação da ONG, missão, visão, contato e imagens |
| Projetos Sociais      | `projetos.html`     | Detalha os projetos, voluntariado e como ajudar |
| Cadastro de Voluntário/Doador | `cadastro.html` | Formulário completo com todos os campos exigidos |

## Especificações Técnicas Obrigatórias (todas implementadas)

- HTML5 100% semântico (`<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<footer>`)
- Hierarquia correta de títulos (`<h1>` → `<h6>`)
- Uso de imagens em todas as páginas
- Formulário complexo com:
  - Nome Completo, E-mail, CPF, Telefone, Data de Nascimento
  - Endereço, CEP, Cidade e Estado
  - Validação nativa HTML5 (`required`, `pattern`, `type="email"`, `type="date"`, etc.)
  - Máscaras automáticas para CPF, Telefone e CEP (via JavaScript puro)
  - Agrupamento lógico dos campos
- Design totalmente responsivo (mobile-first)
- Acessibilidade (contraste, foco visível, alt nas imagens)
- Código limpo, organizado em pastas (`css/`, `js/`)


## Tecnologias utilizadas

- HTML5 semântico
- CSS3 (Flexbox, Grid, variáveis CSS, animações)
- JavaScript vanilla (sem bibliotecas)
- Google Fonts: Montserrat + Playfair Display

## Como rodar o projeto

1. Baixe ou clone este repositório
2. Abra o arquivo `index.html` no navegador
3. Ou acesse diretamente online: https://hyundark.github.io/ong-diamond-dogs/)

> Nenhum servidor é necessário – tudo funciona 100% offline!
